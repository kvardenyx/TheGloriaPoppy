# 🎮 The Gloria Poppy

Веселая собачка кружит по полю, хватая косточки на лету!

---

## 📖 О проекте

Проект был создан в образовательных целях для практики разработки игр. В процессе работы я самостоятельно занимался программированием, созданием графики и изучением полного цикла разработки — от идеи до готового приложения.

Игра не претендует на коммерческий уровень и создавалась как возможность получить практический опыт, попробовать различные подходы к разработке и лучше понять процесс создания игр.

Большая часть проекта, включая код, визуальное оформление и реализацию игровых механик, выполнена мной самостоятельно. Основная идея игры вдохновлена существующим проектом.
